# McGamerZone Public API
To get started, look [here](https://github.com/McGamerZone/PublicAPI/wiki)

## Building From Source
- Clone the project
- Open the project in your IDE of choice
- Run the command **mvn clean install**

## Dependencies
- **lombok** 1.18.16
- **slf4j-simple** 1.7.30
- **gson** 2.8.6
